## 不纳入

| 项目 | 理由 |
|------|------|
| 去重内存泄漏 | 审计确认当前是 DB 查询（`check_dedup` SQL 时间窗口），非内存缓存，不存在泄漏 |
| 插件沙箱化 | 投入大收益低。Docker 部署已隔离，裸机场景 RCE 风险通过文档告知 |
| Source 鉴权 | 泛用转发平台不应限制上游发消息方式 |
